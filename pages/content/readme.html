You've got the hang of it! It appears you've come to the right file! When navigating throughout this website feel free to open any of the files on the right side of the screen. These files contain information related to the current tab you are on. If you want to change pages, click on one of the tabs above this window! As you go through all the files and tabs in this project it should give you a great idea about who I am, what I like to do and overall help you get to know me!

This project was created on 2/21/2025 and finished on 2/22/2025 by Damien Hall. I purposefully styled this to be exotic, bright, bold and crazy to fully embrace the "90's computer geek hacker vibe" aesthetic! Due to 90's aesthetic and general nuance to how websites were created back then, I have purposefully added some "off" characteristics to this site:

  • Bright colors
  • Slightly mis-aligned yet aligned navigation bar
  • Bold blocky text
  • Unstyled text
  • Goofy-not-well-designed images
  • Abnormally large and pixelated cursor
  • Walls of text with no fancy fonts or spacing

While there is alot of stylistic liberty taken here, I have taken the 90's modern by adding some fancier CSS styling and JavaScript functionality to handle certain issues. The most notible styling and functionality choices are as follows:

  • Webpage uses %, em, vw and vh to create scaleability
  • When text on .txt files is too long for the view, it will be trimed with ...
  • Iframes were used to load "file-content" based on the .txt files on the side
  • Images only have max-width and max-height set for image scaleability
